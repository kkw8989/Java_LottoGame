### ⚒️ 기술 스택
- JAVA

### 💡 주요 기능 

<details>
<summary>기능 요약</summary>
<div markdown="1">       
 
- 로또는 1개당 천원으로 지정
- 구매하고 싶은 금액 입력
- 자동, 수동 로또 구매 방식 선택
- 사용자 로또 번호와 랜덤 당첨 번호 비교 후 등수 출력

</div>
</details>
